About

This app acts as an academic tool for students by utilising a recommendation system. The recommendation system uses the user's module info, tasks, previous task history and motivation levels to help to help rank their study tasks.

Tutorial

1. Modules
Click "Edit Modules" → "+ Add Module".
Pick your module out of the currently listed (more will be added in future updates), and set how much you like that module with "Likeness", and how difficult it is with "Difficulty".
You can edit and delete your modules. However, deleting a module will also delete the current active tasks corresponding to that module.

2. Tasks
Note: You must add a module first before adding a task.
Click "+ Add Task". Assign a module, task type, and difficulty to that task. Afterwards, save the task. The recommender system will automatically update.

3. Getting Ranked Tasks
Use the slider to match how motivated you feel at the moment.
Click "Get Tasks" to receive recommendations

4. Complete Tasks
Click "Done" to track progress
